Manuel de l'Agent

Bienvenue sur le Command Center. Cette interface est votre outil de survie et de coordination sur le terrain. Lisez attentivement ces instructions avant votre première expédition.

🗺️ 1. Cartographie et Navigation

La carte est votre atout principal. Vous pouvez changer le style de la carte selon vos besoins :

  • Classique : Idéal pour les repérages de jour.
  • Satellite : Parfait pour repérer les toits, les entrées cachées et la végétation.
  • Sombre (Dark) : Mode tactique de nuit pour ne pas vous éblouir.
  • Topographique : Utile pour les spots en forêt ou montagneux.

📍 2. Signaler un Spot (Ajouter un marqueur)

Pour enregistrer une nouvelle découverte dans la base de données de l'escouade :

  1. Cliquez n'importe où sur la carte à l'endroit exact du lieu.
  2. Remplissez le rapport : Nom du lieu, Type (Hôpital, Usine, etc.), et le niveau de Risque.
  3. Preuve Visuelle : Vous pouvez activer votre appareil photo en direct pour y joindre une image du lieu ou d'un point d'accès.
  4. Cliquez sur Enregistrer. Le point apparaîtra instantanément sur la carte de tous vos alliés.

📡 3. Communication & Escouade

Le système est synchronisé en temps réel avec tous les agents connectés.

  • Radio : Utilisez le canal de chat pour communiquer silencieusement.
  • Traces GPS : Les déplacements de vos alliés sont visibles sur la carte (lignes en pointillés).
  • Mode Hors Ligne : Si vous entrez dans une "Zone Blanche" sans réseau (ex: sous-sol), une bannière jaune vous avertira que vos communications sont coupées.

🚨 4. Protocoles d'Urgence

Bouton S.O.S

En cas de danger immédiat (blessure grave, rencontre hostile, effondrement) :

Pressez le bouton S.O.S. Une alerte visuelle et sonore massive sera envoyée à tous les agents connectés avec vos coordonnées GPS exactes.

Mode Discrétion (Stealth / Fake UI)

Si vous êtes interpellé par des vigiles ou les forces de l'ordre, vous devez protéger les données de l'escouade :

Activez le Mode Discrétion. L'application se transformera instantanément en une fausse application inoffensive de randonnée pédestre ("RandoCartes - Sentiers Publics").

👉 Pour désactiver la fausse application et revenir au Command Center, faites un Double-Clic sur la barre de titre.

Caméra NVG (Night Vision)

Si vous perdez votre lampe torche, vous pouvez activer le bouton NVG. Il allumera la caméra de votre téléphone avec un filtre amplificateur de lumière (vert) pour vous aider à y voir dans la pénombre tout en gardant les mains libres.

Restez prudents. Ne laissez aucune trace. Prenez seulement des photos.
